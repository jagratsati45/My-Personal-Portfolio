## ⚡ Performance Optimizations

This portfolio is engineered for **butter-smooth scrolling** on mobile devices:

```
✅ scroll-behavior: auto on mobile (removes CSS jank)
✅ GPU-promoted layers (translateZ(0), backface-visibility)
✅ requestAnimationFrame-throttled scroll listeners
✅ Passive event listeners for scroll handlers
✅ Particles: 80 → 20 on mobile, 60fps → 30fps cap
✅ Disabled collision detection & hover on touch devices
✅ Removed expensive background patterns on mobile
✅ will-change only during active animations
✅ Reduced box-shadows on mobile for fewer repaints
```
